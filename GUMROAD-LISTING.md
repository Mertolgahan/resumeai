### Stop rebuilding the same infrastructure. Start shipping.

Every SaaS needs auth, payments, AI, rate limiting, and a polished UI. **SaaSKit Pro** gives you all of this out of the box — tested, documented, and ready to deploy.

Built with **Next.js 16 (App Router)**, **Supabase**, **NextAuth v5**, **Stripe**, and **Tailwind CSS**. Just add your idea.

### What You Get

- **Full Source Code** — GitHub repo access, MIT-licensed for personal use, Commercial license for business
- **Production Docker Setup** — Dockerfile, nginx, SSL, deploy scripts included
- **Complete Documentation** — Setup guide, deployment guide, environment configuration
- **Free Updates** — 1 year of updates and new features

---

### Features

**Authentication (NextAuth v5)**
- Google OAuth
- GitHub OAuth
- Email + Password with bcrypt
- JWT sessions, protected routes
- Auto user creation for OAuth

**Payments (Stripe)**
- 4 pricing tiers (Free → Starter → Pro → Enterprise)
- Monthly + yearly billing
- Stripe Checkout + Customer Portal
- Webhook handling (subscription lifecycle)
- Price ID configuration via environment variables

**AI Content Generation**
- 7 content types: Blog Post, Social Media, SEO Title, SEO Description, Product Description, Email Campaign, Ad Copy
- Real-time streaming (SSE)
- Dual-provider: Ollama (self-hosted) + OpenRouter (cloud)
- Smart fallback: tries Ollama first, falls back to OpenRouter

**Usage & Rate Limiting**
- Per-plan daily + monthly limits
- Free: 10/day, 100/month
- Starter: 100/day, 3,000/month
- Pro: 1,000/day, 30,000/month
- Enterprise: Unlimited

**Dashboard**
- Content generation interface
- Generation history with filtering
- Usage analytics (daily/monthly)
- Billing management (upgrade, manage subscription)
- Profile settings

**UI & UX**
- Dark mode with system detection
- shadcn/ui components (20+ components)
- Fully responsive (mobile-first)
- SEO-optimized landing page
- JSON-LD structured data
- Auto-generated sitemap + robots.txt

**DevEx & Deployment**
- TypeScript throughout
- Docker + docker-compose
- nginx reverse proxy config
- Let's Encrypt SSL automation
- Environment-based configuration
